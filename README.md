# clojure-grpc-fizzbuzz

FizzBuzz with gRPC.

## Usage

### Leiningen

    $ lein run

### Docker

    $ docker build -t clojure-grpc-fizzbuzz
    $ docker run -it --rm clojure-grpc-fizzbuzz

### Run tests

    $ lein test
